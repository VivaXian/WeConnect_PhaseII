> 用途：设计师通过Figma制作了部分Phase II关键界面，将这些界面同步给 VS Code / AI 设计模型，以高保真地还原 WeConnect Phase II设计稿。

## 1. Phase II 已有核心页面

### 1.1 「设备」页面

#### 1.1.1 顶部统计视图
Super user 和 普通用户 均可见，只是模块范畴不一样：
Super user可见: 
- 全部设备
- 合同风险 （包括即将出保、已出保设备）
- PM 风险
- 报修中
- 保养计划 （即PM Schedule）

普通用户可见: 
- 全部设备
- PM 风险
- 报修中

顶部模块分展开态和折叠态，进入页面时自动展开，但随着页面向上滚动自动折叠。
本界面参考中提供的是展开态。
院区名称可能很长，要考虑院区名称的选择和展示位置。

#### 1.1.2 设备管理

用户可根据院区筛选设备，院区的来源：
1. super user 授权的院区
2. super user 和 普通用户自己扫码绑定的设备的所属院区；

还可筛选设备类型：
1. CT
2. 磁共振
3. 血管机
