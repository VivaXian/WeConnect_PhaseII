# mini-program top.md

> 用途：给 VS Code / AI 设计模型理解 小程序/mini-program
> 这是小程序顶部固定的一块导航栏区域的figma图，包括状态栏（系统时间，网络连接状态等）和导航栏（小程序的名称，页面名称，返回，退出等操作）。请调用Figma MCP 读取还原。
> 现在给的参考是iOS系统的状态栏
> 导航栏右侧那两个按钮是小程序的功能，左边的···代表设置，右边的圆圈代表退出。这在我们的prototype中不需要作功能演示，只需要还原出样式就可以
